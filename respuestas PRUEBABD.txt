¿Que cliente realizó la compra más cara?
SELECT C.NOMBRE, F.TOTAL FROM cliente AS C INNER JOIN factura AS F ON C.ID=F.ID_CLIENTE WHERE F.TOTAL =(SELECT MAX(F.TOTAL) FROM factura F);
¿Que cliente pagó sobre 100 de monto?
SELECT C.NOMBRE, F.TOTAL FROM cliente AS C INNER JOIN factura AS F ON C.ID=F.ID_CLIENTE WHERE F.TOTAL >100
¿Cuantos clientes han comprado el producto 6?
SELECT COUNT(DISTINCT C.NOMBRE)AS CANT_CLIENTE FROM detalle_factura AS DF
INNER JOIN producto AS P ON P.ID=DF.ID_PRODUCTO
INNER JOIN FACTURA AS F ON DF.ID_FACTURA=F.ID
INNER JOIN CLIENTE AS C ON F.ID_CLIENTE=C.ID
WHERE P.ID=10
